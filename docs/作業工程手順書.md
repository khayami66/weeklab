## 本書の目的
本書は、Weeklab（小学校教員向け週案作成支援Webアプリ）の実装作業を**フェーズごとにチェックボックス化**し、進捗を可視化するための手順書です。各項目を完了するたびにチェックを入れ、フェーズ単位での完了確認ができます。

## 凡例
- `- [ ]` … 未完了
- `- [x]` … 完了
- 太字の見出しは**フェーズ完了条件**（すべて満たしたら次フェーズへ）
