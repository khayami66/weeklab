# screen_design.md

## 1. ルーティング（Next.js App Router）

| パス | 画面 | MVP | 役割 |
|---|---|---|---|
| `/` | ホーム | ✓ | 今日と今週の授業一覧 |
| `/weekly` | 週案生成 | ✓ | 指定週の授業一覧・メモ編集 |
| `/print/weekly` | 週案印刷 | ✓ | 松戸市様式（理科専科版） |
| `/progress` | 進度管理 | ✓ | クラス別進度編集 |
| `/timetable` | 時間割・例外 | ✓ | 基本時間割と特定日の例外 |
| `/settings` | 設定 | ✓ | 学校情報・年度・始業式日・使用パック・年度切替・エクスポート |
| `/monthly` | 月次集計 | ✓ | 教育委員会報告用、月別クラス×単元の実施時数 |
| `/print/monthly` | 月次集計印刷 | ✓ | 月次集計の印刷 |
| `/archive` | アーカイブ | ✓ | 過去年度の週案を閲覧 |
| `/master` | マスタ確認 | ✓ | 年間指導計画・授業内容 |

## 2. 共通レイアウト

### 2.1 構成
- ヘッダー：システム名「Weeklab」／本日日付／**現年度表示**／ナビリンク
- サイドナビ（PC）／ハンバーガー（iPad・モバイル）
- メインコンテンツ
- トースト通知

### 2.2 配色
| 用途 | カラー |
|---|---|
| プライマリ | `blue-600` |
| 強調テキスト | `slate-800` |
| 背景 | `gray-50` |
| 区切り線 | `slate-200` |
| 警告 | `amber-500` |
| 例外（変更マーク） | `rose-500` |
| アーカイブ閲覧モード | `slate-500`（読み取り専用感） |

### 2.3 レスポンシブ
- PC：サイドナビ常時、2〜3カラム
- iPad：1カラム、上部ナビ
- モバイル：1カラム、テーブルはカード化

## 3. ホーム画面 `/`

- 今日の理科予定（時限順カード）
- 今週の予定サマリ（月〜土）
- 今週のヘッダー：第n週、自〇月〇日 至〇月〇日
- クイックアクション：週案 / 進度 / 時間割例外 / 印刷 / アーカイブ

カード表示：時限 / クラス / 単元 / 本時 n/N / 授業内容（1〜2行）

## 4. 週案生成画面 `/weekly`

### 操作
- 週セレクタ：`◀ 前週` / 今週 / `次週 ▶` ＋日付ピッカー
- URLパラメータ `?week=YYYY-MM-DD`

### 表示
- 上部バー：第n週 / 期間 / クラス別集計（週実施・実施累計）
- **週先頭コマ確定エリア**：各クラスの今週最初のコマの単元・本時をプルダウンで確定
  - デフォルト値：`ClassProgress` から推定した次本時
  - 確定後、2コマ目以降は `LessonMaster` 順で自動表示
  - 全クラス分確定した時点で週案が完成（未確定クラスは推定値のまま表示）
- **進度ヘルスバッジ**：各クラスに期待累計との差を表示（amber：±3超、rose：±5超）
- 曜日ごとセクション（月〜土、空日はグレーアウト）
- 各コマ：時限 / クラス / 単元 / 本時 / 授業内容 / メモ
- メモは inline 編集（blur で保存）
- 例外コマに `rose-500` 「変更」バッジ
- 「🖨 印刷」→ `/print/weekly?week=...`

## 5. 週案印刷画面 `/print/weekly`

### レイアウト（Y案・理科専科版）
```
┌────────────────────────────────────────────────────┐
│ 4月13日～4月18日   第2週                            │
│ 検印: 校長( ) 教頭( ) 主幹・教務( ) 担任( )         │
├────────────────────────────────────────────────────┤
│ 曜日 | 日 | 1限 | 2限 | 3限 | 4限 | 5限 | 6限       │
├────────────────────────────────────────────────────┤
│ 月〜土（土も常時表示）                              │
│ 各セル：( クラス ) / 単元名 / 授業内容              │
├────────────────────────────────────────────────────┤
│         │ 3-1│ 3-2│ 3-3│ 4-1│ 4-2│ 4-3│ 4-4         │
│ 週実施  │  2 │  2 │  2 │  3 │  3 │  3 │  3          │
│ 実施累計│ 18 │ 18 │ 18 │ 27 │ 27 │ 27 │ 27          │
└────────────────────────────────────────────────────┘
```

### 印刷仕様
- A4横、1週間1枚
- `@media print` でナビ・ボタン非表示
- 罫線はグリーン系
- 検印欄は空欄（手書き前提）

## 6. 進度管理画面 `/progress`

- クラス一覧テーブル
- 列：クラス / 学年 / 使用パック / 現在の単元（セレクト）/ 完了時数 / 累計時数 / **ヘルスバッジ** / メモ
- **進度ヘルスバッジ**：期待累計との差
  - `ok`：緑（差 < 3時間）
  - `warn`：amber（差 3-5時間）「2時間遅れています」等の文言
  - `alert`：rose（差 ≥ 5時間）
- 「次の単元に進める」ヘルパーボタン
- 超過警告（amber）
- 保存 → トースト

## 7. 時間割・例外画面 `/timetable`

### セクション1：基本時間割
曜日（月〜土）× 時限のマトリクス、MVPは閲覧中心＋簡易編集。

### セクション2：例外設定
- 日付ピッカー → その日の各コマ表示
- 各コマ右にプルダウン：
  - 「変更なし」
  - 「休講」（cancel）
  - 「クラス差し替え」（replace）
  - 「追加」（add）
- 保存 → localStorage ＋ トースト

## 8. 設定画面 `/settings`

### 8.1 基本情報
- 学校名
- 教員名
- **年度**（表示のみ、切り替えは下のボタンから）
- 始業式日（日付ピッカー）
- 使用パック選択（チェックボックス）

### 8.2 年度切り替えセクション
- 「**年度を切り替える**」ボタン
  - モーダル：
    - 新年度（例：2027）
    - 新しい始業式日
    - ☑ 基本時間割を引き継ぐ（デフォルトON）
    - ☑ 使用パックを引き継ぐ（デフォルトON）
  - 「実行」で以下を自動処理：
    - 現年度を凍結アーカイブ
    - 新年度に切替（進度リセット）
- **注意**：進度は全クラス 0 にリセットされる旨の警告表示

### 8.3 バックアップセクション
- 「**JSONエクスポート**」ボタン → 全年度データを1ファイル（`.json`）としてダウンロード
- 「**JSONインポート**」ボタン → ファイル選択 → 全データを上書き
  - 実行前に確認ダイアログ（「現在のデータを上書きします」）

### 8.4 保存
設定保存ボタン → トースト

## 9. アーカイブ画面 `/archive`

### 9.1 年度一覧
- 左サイド：アーカイブされた年度リスト（2025年度、2024年度…）
- 各年度にメタ情報：学校名・教員名・使用パック・総週数・アーカイブ日

### 9.2 週案閲覧
- 年度を選ぶと、その年度の週一覧（第1週〜第n週）
- 週を選ぶと、当時の週案が読み取り専用で表示
- 現在の `/weekly` と同じUI、ただし **編集・メモ保存不可**
- 画面全体にグレーの薄いオーバーレイで「閲覧モード」を明示

### 9.3 印刷
- アーカイブも印刷可能（`/print/weekly?archived_year=2025&week=2`）

## 10. マスタ確認画面 `/master`

タブ切替（閲覧のみ）：
1. **年間指導計画**：学年タブ → 月別一覧
2. **授業内容**：単元選択 → 本時一覧（lesson_no 昇順）

## 11. 月次集計画面 `/monthly`

### 11.1 用途
- 教育委員会報告用の月別実施時数集計
- 頻繁に見る画面ではないが、月末〜月初に必要
- 月〜土の実施時数を月曜日の月基準で集計

### 11.2 表示
- 月セレクタ（`◀ 前月` / 今月 / `次月 ▶`）
- 表形式：行＝クラス（3-1〜4-4）、列＝当該月に学習した単元
  - セル値：その単元でその月に実施した時数
  - 右端列：月合計時数
- 下段：月合計（全クラス合算）

### 11.3 印刷 `/print/monthly`
- A4縦、表形式
- `@media print` でナビ・ボタン非表示
- 学校名・年度・月をヘッダーに記載
